Hexo博客搭建

发表于 2021-12-01 分类于 blog

1.环境装备

  • 安装node.js
  • 安装Git

具体参照：

博客：https://hexo.io/zh-cn/docs/
next主题：https://github.com/theme-next/hexo-theme-next

2.安装 Hexo

$ npm install -g hexo-cli

配置环境变量

初始化blog

$ cd /d E:\studying\blog
$ hexo init blog
$ cd blog
$ npm install --no-fund

编译部署

$ hexo clean  //清理   建议每次都清理一下
$ hexo d -g

服务器

Hexo 3.0 把服务器独立成了个别模块，您必须先安装 hexo-server 才能使用。

$ npm install hexo-server --save

启动服务器，默认是运行在4000端口
$ hexo server
如果您想要更改端口，或是在执行时遇到了 EADDRINUSE 错误，可以在执行时使用 -p 选项指定其他端口，如下：

$ hexo server -p 5000

在浏览器中输入： http://localhost:4000/

3.更改成Next主题

下载next主题(注意所在目录)

$ git clone https://github.com/theme-next/hexo-theme-next themes/next

修改hexo配置_config.yml，把theme改为next

# Extensions
## Plugins: https://hexo.io/plugins/
## Themes: https://hexo.io/themes/
theme: next

重新编译

$ hexo clean  //清理   建议每次都清理一下
$ hexo d -g

Ctrl+C结束服务端，重新运行

$ hexo server

这样就基本上运行起来了，至于具体的Next配置，得看你自己的喜好配置配置文件在 themes/next/_config.yml
