Android studio gradle task list 不显示问题

发表于 2022-04-08 分类于 android

#Android studio gradle task list 不显示问题

问题描述

我在android studio 2020.3.1上面创建的android项目 gradle视图面板没有显示Task列表

###解决办法

  1. File -> Setting -> Experimental 取消勾选 Do not build Gradle task list druing Gradle sync
  1. File -> Sync Project with Gradle Files

最后显示为正常
