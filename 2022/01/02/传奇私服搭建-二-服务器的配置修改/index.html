传奇私服搭建(二) 服务器的配置修改

发表于 2022-01-02 分类于 传奇技术

#1.充值元宝
在D:\MirServer\Mir200\Envir查看MerChant.txt

..........省略很多不相关的内容

;BEGIN 元宝
元宝充值    3    345    338    元宝充值    0    12    0    0    0    0    0
元宝充值    0    330    264    元宝充值    0    12    0    0    0    0    0
元宝充值    ddkq    12    27    元宝充值    0    12    0    0    0    0    0
;END 元宝

元宝充值 3代表的就是盟重的那个充值使者NPC他的名字是后面对应的元宝充值

然后我们去 D:\MirServer\Mir200\Envir\Market_Def文件夹找到元宝充值-3.txt

[@main]
<$USERNAME>您好,欢迎来到<$SERVERNAME>,很高兴为您服务\
您当前持有:<$GAMEGOLD>元宝\
①本系统支持网上银行、手机充值卡、各类游戏点卡。\
②请在充值5分钟后在本NPC处领取\
③当前充值比例为1:10\
\
<领取/@领取>┆<充值/@充值>┆<退出/@exit>\
[@领取]
#CALL [\充值元宝\元宝\元宝.txt] @领取元宝

..........省略很多不相关的内容

可以看到是在\充值元宝\元宝\元宝.txt的**@领取元宝**

我们找到D:\MirServer\Mir200\Envir\QuestDiary\充值元宝\元宝在对应的元宝数的txt文件中填入要充值的游戏账号，如果有赠送啥的也是类似的添加

#2.添加商城的物品

在M2server中的 查看 -> 列表信息 -> 商铺列表
