#魔域私服一条龙版本简单修改教程

一，合BB所需副BB评分如何设定为最高只需19星XO.

打开数据表，找到cq_eudemon_rbn_rqr这张表。然后找到 sacrifice_starlev 这一列。把高于1900的数字改为1900，那么你BB不管多少星。所需要的副BB只需要19星就可以合了。

二，如何让你的BB出生即为高星

打开数据表，找到cq_grade这张表，然后，拉到后面，找到列名为rarity 的竖排，把这里的数字改掉。比如你想要100*的就改10000就可以了。不过不要改的太高，以免不能召唤。

三，BB战改法或法改战

在cq_itemtype下面,找到要改的BB的ID 找到able_mask这栏,把这栏里的改掉好了

7代表可以骑和合体 （非骑宠，改过后没什么用）

3代表合体 （法改战，要拥有战士属性，需要改cq_grade属性）

2就是不可以合体 （战改法，要拥有法师属性，需要改cq_grade属性,同时在cq_action配相关技能）

四，修改合BB的速度（即每次加的评分）

打开数据库，找到名为cq_eudemon_rbn_rqr的表，你会看到列 min 和 max这两列对应都有数字。上面的一般是4000+ 越到下面数字越小你的可能是1或2. 我们如果想要让BB合得快，一般只要把 max里的数字加大即可让你的BB每次评分升高。具体加多少，请自己去试.

五，如何修改合BB不用相同属性

打开cq_eudemon_rbn_rqr表，找到列名为 sacrifice_dmg_type 这里 0代表不需要相同属性， 1为需要相同属性。把1改成0，那么就不受限制了。

六，如何修改BB合星时成功与失败的机率

打开数据库，cq_eudemon_rbn_rqr表，找到列名为 suc_percent 把这里的数字改成100，那合BB不会失败，一般最低改成75就很容易失败了！

七，如何修改天天军团战

打开数据库，找到cq_config这张表。然后找到ID 191（

191 7000 5213000 5233000 21150 0 0 0

把这第三个和第四个的第一个数字 5 改成1就是星期一打，改成3就是星期三打了。

可以从1到7一样的改一个。然后把这表复制出来当成补丁，每天重启服务器时，复制当天日期的替换原来的就好了！

八，修改VIP批量孵化时间

找到cq_config这张表。找到type为3101 到3107的行，ID不说了，有可能你的不同！

3101就是1级VIP。3107就是7级VIP了。 然后把对应的data1 里面的数字改掉（这里1就相当于一秒）改成你想要的时间就OK！

九，修改技能经验及等级

用Navicat这个工具,打开cq_magictype这个表 找到need_exp和need_level这两列 need_exp指升级需要的经验；觉得升的慢可以改的低点。need_level指技能等级提升需要的级别

十，修改升级速度

用Navicat打开数据库找到 cq_levexp 这个表

修改usruplevExp以及euduplevExp(注意两个值需一样)

十一，修改等级上限突破132级

等级上限在cq_config表里面.自己找吧..132改成多少自己说的算..不过极限是255,不要超过这个限制!切记!

十二，魔域自封GM

用navicat打开数据库找到 cq_user 表打开. 这里有你帐号人物的信息.

简简单单的一步 改: 在自己名字后面加上: [PM] 比如： 洒洒水啦[PM]

GM工具也可以实现

十三，修改进入游戏欢迎词

找到客户端里的ini文件夹，里面有个StrRes.ini文件, 找到10052=欢迎进入魔域，魔域的世界有你更精彩…,自己去改吧！（这里***的内容是游戏名字）

十四，设置上线VIP等级

打开数据库,找到account表,选择设计表,然后拉到下面看vip的默认值,0表示普通玩家,1-7的值对应相应的会员等级,自己改吧.

当然GM工具也可以

十五，军团图腾全开

打开数据库，找到cq_syndicate，在里面你会看到已有军团的数据，别的不用管，找到totem_pole这一项，把你要改的那个军团对应的值改为18446744073709551615，这样，你军团的图腾就全开了。

十六，修改魔石商店里物品的方法

打开数据库 找到cq_goods这个表 ownerid为1207的就是魔石商店 itemtype就是魔石商店里卖的东西

出售的价格在cq_itemtype，找到相应物品 这一行最后一个选项就是改价格的地方，有些是0 你直接改成你想卖的价格

但客户端的INI文件夹中的itemtype.dat这个文件也需要修改和服务端对应的价格.

十七，如何修改BB的外形或者怪物，NPC的外型!

修改怪物和BB或者部分含攻击性的NPC， 打开cq_monstertype这张表，找到你想修改的怪物的ID，在第四列名为lookface的行里面，

把他改成你想要变的外型代码（如：439 就是我们常见的神龙王外观代码）

修改BB外型时，有六条相关数据

10710 10711 10712 10790 10791 10792

10710 10790 就是未进化时的外型

10711 10791就是第一次进化后的外型

10712 10792就是第二次进化的外型了。

功能NPC外型的修改，我们打开cq_npc表

选择你要修改的NPC的ID那行，在第五列名为lookface这里，同样修改成你想要他变的外型代码就行了。

修改之前，必须知道你想要改过后外型的代码！

十八，关于城堡系统的修改

1.确保你的魔域城堡系统正常

2.你的电脑能打开数据库（比如说有Navicat 8.0 MySQL）

3.打开魔域数据库里的cq_castle_lev_req，前面几栏都是升级要求，比如人口 金钱什么的 最后一栏是每天可以建造单位的数目，个人建议一定要改，还有人口那一栏也得改，不过建造单位数的上限是255

4.这是最重要的！！

打开魔域ini文件夹里面CastleLevReq.ini这个文件 不是有：

这样的格式吗

money是升级需要金钱 Emoney魔石 Woods木材 Population人口 Item已建单位数

BuildNumEveryday是每天限制建造的单位数

根据你的需要来改

另外 在数据库里面cq_castle这一项是已有的城堡 自己看着改吧 主要是改population和food，记住不要超过max_population和max_food里的数值，在bulid_oper_times这一项里的数值是你当天已建造的单位数

十九，如何实现批量修改暴率

首先用Navicat 打开数据库,然后找到 cq_dropitemrule 表, 这个就是怪物暴率+暴物品的表

点查询,新建查询 输入 update cq_dropitemrule set chance=200000 where chance=500000 点执行即可

暴率为chance这列

200000为要改的暴率 数值的表示: 20万较低 50万中等 100万较高

500000为当前的暴率

item0-item14 都是所暴物品的ID,关于ID代表的物品可以在cq_itemtype表内查询对应ID了解到, 可以自己修改为想要掉的物品 item编号越低越容易出!

下面我们再来看看批量修改所暴物品的范例查询命令:

update cq_dropitemrule set item14=要设置掉的物品ID where item14=当前这列的物品IDitem14为最后一个物品列 可以自己修改item0-item14

好了,去试试吧,不过有两点需要注意;

  1. 在修改前先备份一下数据库,什么,你不知道怎么备份,那就说得简单通俗一点,找到sql server里mysql文件夹下data目录里的my文件夹进行备份(有account文件夹的话也备份一下).

  2. 修改完要记得重新启动服务端程序才能生效.
